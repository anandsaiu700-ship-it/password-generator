# 🔐 CyberPass – Password Generator

A simple yet powerful Python tool to generate **secure and random passwords**.

---

## ✨ Features

* Uppercase, lowercase, numbers & symbols
* Ensures strong password structure
* Randomized & shuffled output
* Basic strength indicator

---

## 🚀 Usage

```bash
git clone https://github.com/anandsaiu700-ship-it/password-generator
cd password-generator
python main.py
```

---

## 📌 Example

```
Enter password length: 10
Generated Password: A@7kL9#pQ2
Strength: Strong
```
